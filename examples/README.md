# Example Workflows

This directory contains example workflows to help you get started with Agent Orchestrator.

## Available Examples

### 1. Research and Summarize
A simple workflow that uses two agents:
- **Research Agent**: Gathers information on a topic
- **Summary Agent**: Summarizes the findings

### 2. Content Creation Pipeline
A multi-step content creation workflow:
- **Idea Generator**: Brainstorms content ideas
- **Outline Creator**: Creates a structured outline
- **Writer**: Writes the full content
- **Editor**: Reviews and refines the content

### 3. Customer Support Assistant
An automated customer support workflow:
- **Classifier**: Categorizes incoming requests
- **Response Generator**: Generates appropriate responses
- **Quality Checker**: Reviews responses before sending

## How to Use

1. Go to the Workflows page
2. Import an example workflow JSON file
3. Customize the agents and connections as needed
4. Run the workflow to see it in action

## Creating Your Own

Use these examples as templates for creating your own workflows. Key tips:

- Start simple with 2-3 agents
- Define clear roles and responsibilities
- Use system prompts to guide agent behavior
- Test with different LLM providers to find what works best
